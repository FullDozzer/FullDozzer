# Telegram-бот для мониторинга расписания колледжа

## Быстрый старт (уже настроено)

Я уже встроил ваши параметры в `bot.py`, поэтому можно запустить без `.env`:

```bash
python -m venv .venv
source .venv/bin/activate  # Windows: .\.venv\Scripts\Activate.ps1
pip install -r requirements.txt
python -m playwright install chromium
python bot.py
```

Этот бот проверяет сайт расписания колледжа и пишет в Telegram, если расписание изменилось.

- Открывает **прямую ссылку расписания группы на завтра** (дата в URL обновляется автоматически).
- Парсит новый формат страницы (`div.header3` + карточки `div.card.myCard`) и отправляет расписание в читаемом виде.
- Парсинг идёт строго по DOM-элементам карточек (`card-header`, `card-body`, `subGroup1`, `subGroup2`).
- Если у одной подгруппы пары нет, бот явно показывает это как `Нет пары`.
- Если на странице написано `Расписание ... не опубликовано`, бот не пытается парсить пары и возвращает понятное сообщение.
- Форматирует расписание и отправляет его как изображение (карточки пар, предмет, аудитория, преподаватель).
> Простыми словами: бот сам открывает сайт, выбирает вашу группу, ставит дату **на завтра** и сравнивает с тем, что было раньше.

---

> ⚠️ Важно: токен бота записан в коде для удобного старта. Лучше потом перенести его в `.env` и сменить токен в BotFather.

## 1) Что нужно заранее

1. **Установить Python 3.11+**
   - Скачайте с официального сайта: https://www.python.org/downloads/
   - При установке на Windows обязательно поставьте галочку **Add Python to PATH**.

2. **Создать Telegram-бота** через `@BotFather`
   - Откройте Telegram → найдите `@BotFather`
   - Команда: `/newbot`
   - Придумайте имя и username
   - Скопируйте токен (вида `123456:ABC...`) — это `BOT_TOKEN`

3. **Написать боту в Telegram**
   - Просто отправьте ему `/start` после запуска — бот сам добавит ваш чат в подписчики.

## 5) Настройка `.env`

1. Скопируйте пример:

```bash
cp .env.example .env
```

На Windows:

```powershell
copy .env.example .env
```

2. Откройте `.env` и заполните:

```env
BOT_TOKEN=ваш_токен_от_BotFather
SCHEDULE_URL=http://www.ishnk.ru/2025/site/schedule/group/508/2026-02-25
GROUP_NAME=точное_название_группы
TIMEZONE=Europe/Moscow
CHECK_INTERVAL_SECONDS=1800
DATE_FORMAT=%d.%m.%Y
IGNORE_HTTPS_ERRORS=true
SCHEDULE_FONT_PATH=
```

Минимально обязательно заполнить: `BOT_TOKEN`, `SCHEDULE_URL` (ссылка на расписание группы), `GROUP_NAME`.

Поддерживается 3 формата `SCHEDULE_URL`:
- Не используйте `https://ishnk.ru/students/schedule` — это старый раздел, бот автоматически переключит на ссылку группы.
- с плейсхолдером даты: `.../group/508/{date}`
- с фиксированной датой в конце: `.../group/508/2026-02-25`
- без даты в конце: `.../group/508`

Бот по умолчанию подставляет **завтрашнюю дату** автоматически. Если расписание на завтра недоступно/пустое, бот отправит **сегодняшнее**.

1. Напишите боту в Telegram: `/start`
2. Затем: `/checknow`
3. Бот ответит:

## Частые проблемы

### 1. `Не нашёл группу ...`
Значит `GROUP_NAME` не совпадает с тем, как группа называется на сайте. Скопируйте название группы **буква в букву**.

### 2. Ошибка браузера Playwright
Переустановите браузер:

```bash
python -m playwright install chromium
```

### 3. Ошибка `Не найдены обязательные переменные окружения` или `KeyError: BOT_TOKEN`
- Создайте файл `.env` рядом с `bot.py` (командой `copy .env.example .env` или `cp .env.example .env`)
- Заполните минимум: `BOT_TOKEN`, `SCHEDULE_URL`, `GROUP_NAME`
- Запускайте `python bot.py` из папки проекта

### 4. Бот молчит
- Проверьте `BOT_TOKEN`
- Убедитесь, что вы написали боту хотя бы 1 сообщение


### 5. Ошибка `ERR_CERT_AUTHORITY_INVALID`
Это проблема SSL-сертификата сайта (часто на старых серверах).
- В `.env` поставьте: `IGNORE_HTTPS_ERRORS=true`
- В текущей версии это уже включено по умолчанию.

### 6. В Telegram приходит `404 Not Found`
Теперь бот не отправляет страницу 404 как расписание. Он покажет понятную ошибку: расписание на эту дату ещё не опубликовано.


### 7. В картинке вместо текста квадратики
Это проблема шрифта на системе.
- Укажите путь к шрифту с кириллицей в `.env`: `SCHEDULE_FONT_PATH=C:/Windows/Fonts/arial.ttf`
- Linux пример: `SCHEDULE_FONT_PATH=/usr/share/fonts/truetype/dejavu/DejaVuSans.ttf`

### 8. Ошибка `BrowserType.launch: Timeout ...`
Бот теперь умеет резервно загружать страницу без Playwright (через обычный HTTP-запрос), поэтому даже при проблемах запуска Chromium расписание должно приходить.
Если ошибка повторяется — проверьте доступ к сайту из браузера и firewall/антивирус.

## Команды бота
- `/start` — показать статус и подписаться на уведомления
- `/subscribe` — подписаться на рассылку
- `/unsubscribe` — отписаться от рассылки
- `/schedule` — показать расписание (по умолчанию на завтра, если пусто — на сегодня)
- `/checknow` — немедленно проверить изменения (если изменений нет, отправит сегодняшнее расписание)
- `/scheduletext` — получить расписание обычным текстом
- `/date ДД.ММ.ГГГГ` — получить расписание на конкретную дату (пример: `/date 25.02.2026`)
- Просто отправьте дату в чат в формате `ДД.ММ.ГГГГ` — бот пришлёт расписание за этот день
- Кнопка `📝 Получить расписание текстом` — быстрый текстовый запрос
- Кнопка `📅 Расписание по дате` — подсказка по формату даты

## Для вашего сайта может понадобиться донастройка

У колледжей разные сайты расписания. Если структура сайта нестандартная, нужно будет поправить селекторы в функции `fetch_schedule_data()` в `bot.py`.
